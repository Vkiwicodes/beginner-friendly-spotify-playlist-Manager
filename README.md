 ##🎵 Spotify Playlist Manager
 A simple command-line spotify 
 Playlist Manager built using Python.

 This project allows users to create and manage a playlist by adding, viewing, searching, and removing songs.

 ## Features
 - Add songs
 - View playlist
 - Remove Playlist
 - Search songs
 - Display the total number of songs
 - Exit the program

 ## How to Run
 when the program asks, enter your choice:
 - **1** to Add a song
 - **2** to View Playlist
 - **3** to Remove a Song
 - **4** to Search for a song
 - **5** to display Total Songs
 - **6** to Exit

Then enter the song name when prompted.
The program performs the selected operation and displays the result.

## Files
- **main.py**- Contains the spotify Playlist Manager program with functions for adding, viewing, removing, and searching songs.

## About
This project was created while learning python fundamental- lists, functions, loops, conditional statements, and user input.
